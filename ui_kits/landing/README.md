# NeuroConvert — Landing Page UI Kit

Public acquisition page. Communicates scientific authority through score demo,
social proof, and precise copy. Single scrollable page with sticky nav.

## Screens / Sections
- **Hero** — Headline, score demo animation, CTA
- **Social proof** — Client logos + key metrics
- **How it works** — 3-step process
- **Score showcase** — Example laudo preview
- **Pricing** — 3 tiers (Free, Pro, Enterprise)
- **Footer** — Nav + legal

## Components
- `LandingNav.jsx` — Sticky frosted-glass top nav
- `LandingHero.jsx` — Hero with animated score gauge
- `LandingHowItWorks.jsx` — 3-step explainer
- `LandingScoreDemo.jsx` — Example laudo card
- `LandingPricing.jsx` — Pricing cards (3 tiers)
- `LandingFooter.jsx` — Footer

## Design Width
1280px — desktop-first marketing page
