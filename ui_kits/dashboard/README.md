# NeuroConvert — User Dashboard UI Kit

Main app interface. Shows laudos history, individual report detail,
score overview, and new analysis flow.

## Screens
- **Overview** — Score summary cards, recent laudos list
- **Laudo Detail** — Full report: gauge, category bars, findings, recommendations
- **Nova Análise** — URL input + analysis loading state

## Components
- `DashShell.jsx` — App shell: sidebar + topbar + content area
- `DashHome.jsx` — Overview screen
- `DashLaudoDetail.jsx` — Full laudo report screen
- `DashNewAnalysis.jsx` — New analysis form + loading state

## Design Width
1280px — desktop app, fixed sidebar layout
