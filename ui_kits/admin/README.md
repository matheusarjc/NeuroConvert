# NeuroConvert — Admin Panel UI Kit

Internal tool for the NeuroConvert team. Financial metrics, user management,
Stripe subscription data, MRR/churn dashboards.

## Screens
- **Overview** — MRR, churn, active users, recent signups
- **Users** — Searchable user table with plan + usage data
- **Subscriptions** — Stripe subscription list with status

## Design Width
1280px — desktop-only internal tool
